# 分享图片目录

此目录用于存放小程序分享相关的图片资源。

## 分享卡片图片

- **文件名**: `share-card.png`
- **尺寸**: 500x400px (5:4 比例)
- **格式**: PNG 或 JPG
- **大小**: 建议小于 200KB

## 使用方法

1. 使用项目根目录下的 `share-card-generator-auto.html` 生成分享图片
2. 将生成的图片重命名为 `share-card.png` 并放到此目录
3. 在代码中使用路径：`/images/share/share-card.png`

## 注意事项

- 图片路径使用绝对路径（以 `/` 开头）
- 确保图片文件已添加到版本控制中
- 如需更新图片，替换文件即可，无需修改代码
